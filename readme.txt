`Post Views Summary` is a dashboard plugin that shows the most and least visited posts and pages.

== Description ==

`Post Views Summary` is a free configurable dashboard plugin.

= There are the following configuration options: =

* Choose between posts, pages and both
* Choose the maximum number matches to display
* Chose between the most and the least read
* Choose between the limitation of tags (selectable), categories (selectable) and unlimited

== Installation ==

1. Upload the `Post Views Summary` plugin (the whole post-views-summary folder) to the `/wp-content/plugins/` directory.
2. Activate the plugin through the *Plugins* menu in WordPress.
3. Set the options for the plugin

== Frequently Asked Questions ==

= Are there some requirements for this plugin? =

You need to install the [WP-PostViews Plugin](http://wordpress.org/extend/plugins/wp-postviews/)

= Where is the configuration data stored? =

The configuration data is stored in the WordPress MySQL `_options` Table. The relevant `option_name` field had the contents `PostViewsSummary`

= Where can i set the options? =

When you drag the mouse over the upper right corner of the `Post Views Widget` you will find the 'Configure' link.
